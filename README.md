# Example Library Cheat

Projeto para pessoas que querem fazer cheat em livrarias apenas. Sem flutuante e etc.

#### Compilando em
- [X] Android Studio
- [X] AIDE  ``armeabi-v7a``
- [ ] AndroidIDE

#### Para iniciar a lib

```
const-string v0, "YourLibName"

invoke-static {v0}, Ljava/lang/System;->loadLibrary(Ljava/lang/String;)V
```
